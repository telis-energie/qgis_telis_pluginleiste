# qgis_telis_pluginleiste – Projekt-Kontext

QGIS 3.x-Plugin: **Telis Pluginleiste**
Erstellt und verwaltet die gemeinsame `TelisToolbar` für alle Telis-QGIS-Plugins.

## Stack

- PyQGIS (QGIS 3.x), Python (OSGeo4W)
- Minimal-Plugin ohne Dockwidget

## Build

```bash
pyrcc5 resources.qrc -o resources.py
```
